DDCM
====

Database Data Collection and Monitoring
This is the repository for the DDCM ruby system. It is a stand alone or centrally managed data collection and monitoring system for databases - mainly oracle.

## Requirements:
* Ruby 1.9 or higher
* OCI8
* XMLSimple
* webrick
* uri
* socket
* resolv
* open3
* json
* date
* fileutils

More coming later.
